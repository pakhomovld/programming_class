This is a notebook-style repository where I will be documenting all my programming lessons.

I write all my lessons in one language and then translate them into another. For example, I first write them in Python and then translate the code into Go. I do this so I don't have to think in terms of just one language. Also, when translating, I encounter errors for the first time that need to be resolved. I think I learn faster and more effectively this way, and I also think knowing a second language is never a bad thing.

## Repository Structure

The repository is organized by DSA theme. Open **[dsa_map.html](dsa_map.html)** in a browser for a visual coverage map.

### sorting/
Sorting algorithms: bubble sort, gnome sort, selection sort, merge sort, quick sort.
**Languages:** Python, Go

### recursion/
Recursion fundamentals and patterns: factorial, permutations (backtracking), Tower of Hanoi, recursive math operations, recursive filtering, string unpacking.
**Languages:** Python, Go

### strings/
String manipulation and algorithms: low-level string utilities in C (length, concat, reverse, case conversion, palindrome, substring search), run-length encoding, palindrome validation.
**Languages:** C, Python

### arrays/
Array techniques: longest monotonic subarray, in-place element removal, summary ranges.
**Languages:** Python

### game_theory/
Game AI: tic-tac-toe with minimax algorithm, game state evaluation, winner detection.
**Languages:** Python
